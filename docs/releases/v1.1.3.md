## What changed

- Added **Clean Logo**, a dedicated pipeline for simple transparent graphics.
- Restricted Clean Logo layers to opaque colors sampled from the source image.
- Added support, local-regression, and final edge audits to reject stray colors, black fringes, and source-absent lines.
- Kept Beam Clean as the default and left the existing complex-image pipelines unchanged.

## Downloads

- **`GTAEmblemMaker-v1.1.3.exe`** — recommended for most users. It extracts a validated, versioned runtime under `%LOCALAPPDATA%\GTAEmblemMaker\app` and launches the application without administrator rights.
- **`GTAEmblemMaker-v1.1.3.zip`** — portable package containing the same application and runtime files.

## Requirements and tradeoffs

- Windows 10 version 1809 or newer, x64, and .NET Framework 4.8.
- An NVIDIA CUDA-capable GPU.
- Clean Logo favors edge correctness over filling every small color variation.
- The application is unsigned, so Windows SmartScreen or antivirus software may display a warning.
