## What changed

- Fixed **Official Catalog Quality** exports so its curves and round shapes use the save-compatible geometry and serialized SVG paths accepted by Rockstar's editor.
- Set the production import-code budget to decimal **1,250,000 characters** for more reliable Social Club imports.
- Removed the misleading **Best Quality** label. **Beam Clean** is the default, while Perceptual AlexNet 224 and Official Catalog Quality remain source-dependent alternatives.
- Reduced the production package to three clearly named fitting strategies.
- Added a self-extracting Windows EXE while keeping the portable ZIP.

## Downloads

- **`GTAEmblemMaker-v1.1.2.exe`** — recommended for most users. It extracts a validated, versioned runtime under `%LOCALAPPDATA%\GTAEmblemMaker\app` and launches the application without administrator rights.
- **`GTAEmblemMaker-v1.1.2.zip`** — portable package containing the same application and runtime files.

## Requirements and tradeoffs

- Windows 10 version 1809 or newer, x64, and .NET Framework 4.8.
- An NVIDIA CUDA-capable GPU.
- Complex-image fitting is compute-intensive and can take a long time. This release favors reconstruction quality over instant results.
- The application is unsigned, so Windows SmartScreen or antivirus software may display a warning.
