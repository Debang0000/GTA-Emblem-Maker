# GTA Emblem Maker v1.1.2 Catalog Export Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use `superpowers:executing-plans` to implement this plan task by task.

**Goal:** Make Official Catalog output saveable by Rockstar while preserving the v1.1.1 fitting behavior, raise every production budget to 1,270,000, and produce an unpackaged v1.1.2 test build.

**Architecture:** Keep `OfficialCatalog.CurveBasis` and `RoundBasis` as the high-precision fitting registry used by the CPU/CUDA search path. Add a second canonical export registry containing Rockstar's current path text and integer dimensions, then select that registry only in `Shapes.ToExportShape`. This keeps search geometry stable while changing the final SVG, layer data, and payload preview together.

**Tech Stack:** C#/.NET Framework 4.8, JSON production profiles, native CUDA scorer, PowerShell build scripts.
